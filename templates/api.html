The Dad Joke Api

How to use the API

The API url down below returns a random dad joke on each request, in JSON format.

To add a new joke to the list just submit a pull request to the jokes.txt file in the GitHub repo. The database.py file adds the new lines in the txt file to the list of possible jokes.
